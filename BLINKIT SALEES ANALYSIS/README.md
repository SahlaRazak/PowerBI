# Blinkit Sales Analysis Dashboard – Power BI

## Objective
  Analyze Blinkit’s sales data to identify patterns in customer behavior, outlet performance, and product trends, enabling data-driven business decisions.

## Dataset
- ~8,500 sales records  
- Contains details on:
  - Item Type  
  - Outlet Size  
  - Outlet Location Type  
  - Outlet Establishment Year  
  - Ratings, Sales and Order metrics  

## Tools Used
   - **Power BI** – Data modeling, DAX calculations, dashboard creation  
   - **Power Query** – Data cleaning and transformation
   - **DAX** – KPI calculations and business metrics
   - **Bookmarks & Slicers** – Interactive navigation between Home and Matrix views

## Data Preparation
- Cleaned missing and inconsistent values using Power Query  
- Standardized categorical columns for Outlet Size, Item Type and Location Type  
- Created DAX measures for:
  - Total Sales  
  - Average Sales  
  - Order Count  
  - Average Rating
 
## Dashboard Highlights
  - **KPI Cards** for quick performance tracking:
          **1) Total counts**
          **2) Average Sales**
          **3) Total Sales**
          **4) Average Rating**
 - **Interactive Filters (Slicers):**
          **1) Outlet Location Type**
          **1) Outlet size**
          **1) Item Type**
   
## Key Insights
- Tier-3 outlets generated the highest total sales compared to Tier-1 and Tier-2.  
- Medium-sized outlets contributed the largest share of revenue.  
- Fruits & Vegetables and Snack Foods are the top-selling item categories.  
- Tier-2 outlets show high average order value but lower volume, highlighting strong expansion potential.  
- Sales peaked around 2018, indicating a rapid growth phase before stabilizing.
